跑者鉴定

为进一步测试和确认耐克运动研究实验室的发现，耐克与英属哥伦比亚大学运动医学研究基金会（BCSMRF）合作进行了一项外部研究。

在12周的时间里，BCSMRF让226名跑者对新款NIKE REACT INFINITY RUN跑鞋进行了测试。在测试中，跑者分别穿着NIKE REACT INFINITY RUN跑鞋和NIKE STRUCTURE 22跑鞋，进行了60,000英里的试验，测试跑者穿着两双不同跑鞋后的伤病发生率和疼痛感。（伤病定义：由于跑步相关伤痛，跑者连续3次或3次以上停止跑步。）

研究结果证实了耐克的发现：经过12周的训练，测试结果显示，穿着NIKE REACT INFINITY跑鞋的跑者的伤病发生率有明显的降低，且跑者们认为他们的膝盖和脚部疼痛程度有所减轻。

2020年，第二次测试中，使用了相同的逻辑设定，测试了NIKE REACT INFINITY 2 和 NIKE ZOOMX INVINCIBLE 跑鞋，相较于NIKE REACT INFINITY的测试结果，没有展示出明显的差异性。

耐克对迄今取得的成果感到非常兴奋，科研团队认为这是为跑者创造更加光明的无伤未来而迈出的又一步。耐克将持续在预防伤病领域进行研究、迭代和扩展，并继续与精英跑者和日常跑者合作。

科学制定多样化训练

从运动基本素质或基本运动能力开发的角度来说，就不是单一的，因此想要自己能够在跑步项目上的运动表现有所提升，就需要从所涉及的多种基本素质（基础运动能力）的训练出发。通常情况下基本素质包括：力量、速度、耐力、灵活性、协调性等，也有部分理论会加入平衡性、敏捷性、柔韧性等。因此在训练的过程中需要将训练种类进行多样化组合。

不同动作和强度组合，更有利于打造强壮的体魄，不同种类的跑步训练则更有利于速度、力量及耐力的提升，同时又能对过度使用性伤害起到有效预防。

对硬核跑者来说，训练的过程就是疲劳—恢复—再疲劳—再恢复的过程，因此训练会产生身体的疲劳，当身体疲劳不能及时消除的时候就可能会影响运动能力的提升，甚至造成运动伤痛或伤病。通常情况下，在训练中所使用最多的运动性恢复手段即为积极性休息，指运动过程中为了消除疲劳而采取的各种变换动作或运动强度的练习。持续时间较短、强度较低的恢复性跑步是跑步训练中最为常见的运动性恢复手段，这样的运动情况下，选择鞋底具备优秀缓冲性和支撑性的鞋款会是比较好的选择，REACT INFINITY 2就同时具备这两种基本的性能，将会是在类似训练过程中的适合鞋款。

而对于入门的跑者来说，身体的各项机能的开发程度还是初始阶段，因为短时间低强度的跑步训练不会给身体短时间内造成极大的负荷，因此对于跑步的入门选手来说，轻松的慢跑可以更好的满足他们的需求。

对于硬核跑者和入门跑者来说，慢跑从训练本质上对于两者并没有区别，但是训练的目的和距离强度却有所不同。

- 耐力跑

长距离训练，低配速，旨在打造肌肉的耐力，同时磨练跑者的意志。

频率:通常每周一次

- 基础性跑

中距离训练，速度适中，帮助跑者累积日常跑量。

频率:一周两到三次

- 间歇跑和法特莱克训练跑法

高强度的短距离跑与一定时长的休息恢复交替进行， 通过短时间内的爆发以提升速度与耐力。间歇跑一般会在田径场上进行精确距离的训练。

频率:每周一到两次

- 节奏跑

中长距离，按照乳酸阈值对应强度进行训练，可以帮助有氧耐力能力的提升，从而可以作用于马拉松项目跑的更快，跑的更远。

频率:每周一到两次

- 恢复跑

这类跑步训练通常是在大量消耗肌力的比赛后、计时赛后或高强度训练后进行，它们距离短、强度轻，意在让身体有时间恢复。

频率:根据实际情况而定

跑者的心理硬核

无论是一次训练课，一场马拉松，还是人生漫漫长路，太多不确定的外因会影响我们的表现：突如其来的大风天、原定路线封闭导致的变化、伙伴受伤缺席，甚至是到场助威的亲友团，一切都可能会带来心态和状态上的变化。为一次奔跑进行准备时，我们在准备的一切其实都是为了帮助自己更好的“掌控”这一段奔跑：更舒适的鞋子应对不可控的路面，更专业的装备应对不可控的天气，更科学的运动饮料配比、配速方案来应对不可控（但会随着训练慢慢变可控）的身体状态。

在职业跑者的心理训练中，积极思维的训练核心就是提高对于赛场的“控制感”。英国长跑名将莫法拉赫曾在世锦赛由于过度兴奋出现过战术错误。在备战伦敦奥运会时，为了避免再次失控，他用一年多的时间进行心理训练，想象在奥运决赛场可能经历的观众欢呼与主场压力以及自己应如何展现最佳状态以获得冠军。这种想象训练的目的是为了让彼时彼刻的情绪可控，不至发挥失常，最终他如愿在故乡夺冠。对于普通跑者来说，来自冠军的经验是：分解目标，制定计划，积极想象，总之要用一切可以获得控制感的方法去武装自己的思维。

一个好的准备方案定是引导跑者充分利用自己可以控制的因素，这也是最为积极的心理定势：以我为主。纠结于不可控的因素会带来焦虑和恐惧，而聚焦于可控的解决方法则带来踏实感。在每次训练前、比赛前，形成属于自己的个性化参赛方案，写下自己的可控因素、优势和最近的进步，把自己的心态专注在“自己可以控制的地方”，不要自我放大那些不可控的地方，给自己徒增不必要的恐惧和紧张感，从心态上就距离成功又近了一步。当这样积极的思维形成习惯，便拥有了一颗硬核的跑者之心。

让日常的运动复原

行之有效

跑步是一项让人着迷的运动，跑步后的肌肉酸爽给我们的身心带来快乐。不过跑步运动同样会给身体带来压力，甚至受伤的风险。但当身体出现反应时，我们首先要有意识去感受身体给予的反馈，同时正确分辨是运动后的正常肌肉酸痛反应，还是伤病带来的疼痛。

通常跑步后的肌肉反应是暂时的，最多持续2-3天后会减弱或消失；并且酸痛主要来源于肌肉，有时会让你感觉到酸胀、疲劳、紧张等。但是跑步受伤造成的疼痛有可能会持续一段时间甚至会加重，它的感觉会有肿胀、刺痛或剧痛等，疼痛来源也会包括关节、肌腱、韧带。同时，要相信通过科学的恢复性训练及积极的日常防护是可以避免运动伤病，健康、快乐地跑步。

更重要的是，对跑者的防护应该是在日常，而不是在伤病发生后。

如果判断是运动后肌肉酸痛

我们可以通过几种方式缓解:

- 避免背靠背跑步训练，降低训练量及强度。

- “积极的休息”：通过其他运动比如游泳，蹬车，椭圆仪等等代替跑步。

- 冰敷，每次10-15分钟， 一天2-3次。

- 可以用手或者筋膜枪轻度按摩松解酸痛肌肉。

- 避免过重过深地按摩。

- 在下次跑步运动中，适当调整跑步距离及速度。

如何在日常跑步运动中

减少伤病发生率，其中包括：

- 跑步前

定期加强整下肢及核心肌力、稳定性训练，建议一周至少1次综合性力量及核心稳定性训练。

适度增加跑步训练量及强度（每周不超过10%的提升）。

避免过多地尝试上下坡冲刺或者高强度间歇跑的训练。（如果有陈旧伤病，需更加注意跑步训练距离及强度的调整，避免运动后的疼痛发生。）

在跑步运动前进行动态热身及牵拉， 避免只做静态拉伸。

- 跑步中

在跑步运动中，跑步速度应循序渐进提升，避免在运动开始就全力冲刺。

- 跑步后

给予自己充分的时间做静态拉伸，在跑步运动后适当地牵拉，避免肌肉紧张，降低拉伤风险。

如跑步运动后有肌肉酸痛，可以通过冰敷，轻度按摩，以及其它低强度运动来缓解消除。
